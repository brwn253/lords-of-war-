                    LORDS OF WAR - CARD GAME
═══════════════════════════════════════════════════════════════

QUICK START
───────────
1. Open "lords-of-war.html" in any web browser (Chrome, Firefox, Safari, Edge)
2. Click "Single Player" to start a game
3. Enter your player name and select a theme color
4. Choose your unit type (Ranged, Infantry, or Cavalry)
5. Select your hero and start playing!

GAME FILES NEEDED
──────────────────
✓ lords-of-war.html       (Main game file - open this)
✓ lords-of-war.js         (Game data and constants)
✓ lords-of-war-engine.js  (Game logic and mechanics)

Make sure all three files are in the same folder!

HOW TO RUN
───────────
Option 1 (Easiest):
  - Double-click "lords-of-war.html"

Option 2 (Using Browser):
  - Open your browser
  - Press Ctrl+O (or Cmd+O on Mac)
  - Select "lords-of-war.html" from this folder

Option 3 (Using File Explorer):
  - Right-click "lords-of-war.html"
  - Select "Open with" and choose your browser

GAME OVERVIEW
──────────────
• Reduce your opponent's health to 0 to win
• Each unit type has unique abilities:
  - Ranged: Don't take damage from melee attacks
  - Infantry: Hero starts with a weapon
  - Cavalry: All units can attack immediately (Charge)

• Play cards from your hand to battle the enemy
• Duplicate units to increase their tier and power
• Use hero abilities to draw cards (costs 2 gold)

CONTROLS
─────────
• Click cards in your hand to play them
• Click units on the battlefield to select them for attack
• Click "End Turn" to end your turn
• Click "⚙️" (top right) during game for settings menu

THEME & CUSTOMIZATION
──────────────────────
• Select your theme color before starting (Blue, Red, Green, Purple, Gold)
• Enter your player name and choose a flag
• Your theme color box appears on the main menu

TIPS FOR PLAYING
──────────────────
• Plan your card plays strategically
• Build up your board gradually
• Don't waste gold on cards you don't need
• Use hero ability when your hand is low
• Watch out for the AI's unit strategies!

SYSTEM REQUIREMENTS
────────────────────
• Any modern web browser (Chrome, Firefox, Safari, Edge, etc.)
• No installation required
• No internet connection needed
• Works on Windows, Mac, and Linux

TROUBLESHOOTING
─────────────────
Issue: Page won't load
→ Make sure all three .js files are in the same folder as the .html file

Issue: Cards or buttons aren't visible
→ Try refreshing the page (F5 or Ctrl+R)
→ Zoom reset: Press Ctrl+0 (or Cmd+0 on Mac)

Issue: Game freezes
→ Refresh the page and try again
